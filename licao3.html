Lição 3: Estruturas Condicionais

As estruturas condicionais permitem que seu programa tome decisões com base em condições. Aqui estão algumas das estruturas condicionais mais comuns:

  • if: Executa um bloco de código se uma condição for verdadeira.
  • else: Executa um bloco de código se a condição do if for falsa.
  • else if: Adiciona condições adicionais às estruturas if.
let numero = 10;

if (numero > 0) {
    console.log("O número é positivo");
} else if (numero < 0) {
    console.log("O número é negativo");
} else {
    console.log("O número é zero");
}
