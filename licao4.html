Lição 4: Laço For

O laço for é utilizado para repetir um bloco de código um número específico de vezes. Ele é muito útil quando sabemos com antecedência quantas vezes queremos executar um bloco de código.

Exemplo:

for (let i = 0; i < 5; i++) {
    console.log("O valor de i é: " + i);
}
